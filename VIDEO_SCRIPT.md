# Cookie Editor Pro — 25-second screencast script

No voiceover needed; on-screen actions tell the story. Record the popup at a real site (e.g. youtube.com).

| Time | On screen | Action |
|------|-----------|--------|
| 0–3s | Open the popup on a busy site | Click the 🍪 icon; show the cookie list for "This site" |
| 3–7s | Domain grouping | Click **All**, scroll the grouped domains, expand/collapse one |
| 7–11s | Search | Type a name in the search box; show instant filtering |
| 11–16s | Edit a cookie | Click a cookie → change the value → **Save** (toast appears) |
| 16–19s | Protect | Click 🔒 **Protect** on a cookie; show the lock badge in the list |
| 19–23s | Settings / auto-clean | Open ⚙ → show Block list (`*.doubleclick.net`) + Allow list → **Run clean-up now** |
| 23–25s | End card | Show the icon + "Cookie Editor Pro — 100% local" |

Tips:
- Resize the window so the popup is large and crisp.
- Move the mouse deliberately and pause ~1s on each result.
- Export at 1080p; ≤60s; upload to YouTube (Unlisted is fine).
