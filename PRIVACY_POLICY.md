# Privacy Policy — Cookie Editor Pro

_Last updated: 30 June 2026_

**Cookie Editor Pro does not collect, store, transmit, or sell any personal or user data.**

## What the extension does
Cookie Editor Pro lets you view, search, edit, add, delete, protect, back up, and auto-clean the cookies stored in your own browser. All of this happens **locally on your device**.

## Data collection
- We do **not** collect any personal information.
- We do **not** use analytics, telemetry, trackers, or advertising.
- We do **not** send your cookies, browsing data, or any other information to any server. The extension makes **no network requests** of its own.
- We do **not** have any account, login, or backend service.

## Data storage
The only data the extension stores is kept **locally** in your browser via Chrome's `storage` API, and never leaves your device:
- Your list of "protected" cookies.
- Your allow-list / block-list patterns and auto-clean preferences.

You can remove this data at any time by clearing the extension's storage or uninstalling it.

## Permissions
- **cookies / host access** — to read and modify the cookies you choose to manage. Used only for the extension's core function; cookie data is never transmitted.
- **tabs** — to read the active tab's URL so cookies can be scoped to the current site (and for the optional "clear on tab close" feature). No browsing history or page content is collected.
- **storage** — to save your settings locally, as described above.

## Remote code
The extension does **not** load or execute any remote code. All code is contained in the published package.
